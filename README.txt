EMBI Country Risk Dashboard
===========================

Instructions:
1. Double-click 'start_dashboard.bat' to launch the dashboard in your browser.

Updating Data:
1. If you have a new version of 'Serie_Historica_Spread_del_EMBI.xlsx', place it in this folder or the parent folder.
2. Run 'python convert_data.py' to update 'data.json'.
   (You need Python and pandas installed: pip install pandas openpyxl)
